## 1. 개요

### 제안 요약

EconoBridge AI는 구직자가 자연어로 대화하면 AI 에이전트가 **7개 공공데이터 API를 자율적으로 호출**하여 채용정보 검색부터 기업 안전성 교차검증까지 한 번에 수행하는 **고용안전 자율 에이전트**입니다.

예를 들어 구직자가 *"경기도에서 IT 개발자 채용 찾아줘"*라고 입력하면, AI 에이전트가 다음 과정을 자동으로 수행합니다:

1. **채용정보 검색** — 워크넷 API에서 조건에 맞는 채용공고 탐색
2. **기업 산재이력 조회** — 근로복지공단·고용노동부 데이터로 해당 기업의 산업재해 이력 확인
3. **안전점수 산출** — 산재 건수·중대재해·사망사고·임금체불 이력을 종합하여 100점 만점 안전점수 자동 계산
4. **위험기업 경고** — 안전점수가 낮은 기업은 명확히 경고 표시
5. **안전한 일자리 추천** — 안전점수 기준으로 정렬된 맞춤 채용 결과 제공

**핵심 차별점**: 기존 채용플랫폼이 급여·복리후생 중심으로 정보를 제공하는 것과 달리, EconoBridge AI는 **"이 회사는 안전한가?"**라는 질문에 공공데이터 기반의 객관적 답을 제공합니다. 단순 채용검색이 아닌 **기업 안전성 교차검증의 자동화**가 본 제안의 핵심입니다.

## 2. 추진배경

### 2-1. 산업재해 현황과 구직자 보호의 사각지대

우리나라는 OECD 국가 중 산업재해 사망률이 여전히 높은 수준에 머물러 있습니다. 고용노동부 산업재해 통계에 따르면 매년 수백 명의 근로자가 산업현장에서 목숨을 잃고 있으며, 2022년 중대재해처벌법 시행 이후에도 중대산업사고는 지속적으로 발생하고 있습니다.

그러나 **구직 단계에서 기업의 안전성을 사전에 확인할 수 있는 수단은 사실상 전무**합니다. 구직자는 급여, 근무시간, 복리후생 정보는 쉽게 비교할 수 있지만, 해당 기업이 과거 산업재해를 얼마나 일으켰는지, 중대재해가 발생한 적이 있는지는 알기 어렵습니다.

### 2-2. 공공데이터의 분산과 활용 부족

기업의 안전 관련 정보는 이미 공공데이터로 존재합니다:
- **근로복지공단** — 산재처리현황 공개데이터
- **고용노동부** — 중대산업사고 발생현황, 임금체불 명단
- **한국산업안전보건공단** — 안전보건법령 정보

그러나 이들 데이터는 각기 다른 기관의 서로 다른 시스템에 분산되어 있어, 일반 구직자가 이를 통합적으로 조회·비교하는 것은 현실적으로 불가능합니다. **데이터는 있으나 활용되지 못하는 "정보 비대칭" 상태**에 놓여 있습니다.

### 2-3. AI 기술의 성숙 — 자율 에이전트의 등장

최근 대규모 언어모델(LLM) 기반 AI 에이전트 기술이 급속히 발전하면서, AI가 사용자의 자연어 요청을 이해하고 **복수의 외부 API를 자율적으로 호출하여 결과를 종합하는 것**이 가능해졌습니다(ReAct 패턴). 이를 통해 분산된 공공데이터를 하나의 대화 인터페이스로 통합 활용할 수 있는 기술적 기반이 마련되었습니다.

### 2-4. 재단 공익사업과의 연계

산업안전상생재단은 **중소기업 안전역량 강화**를 핵심 미션으로 수행하고 있습니다. EconoBridge AI는 다음 두 가지 측면에서 재단 사업과 직접 연계됩니다:

- **구직자 보호**: 위험한 기업을 사전에 식별하여 구직자가 안전한 일자리를 선택할 수 있도록 지원
- **기업 안전개선 유도**: 안전점수의 가시화를 통해 기업 스스로 안전수준을 개선하려는 동기를 부여하고, 안전점수가 낮은 기업을 재단 컨설팅·교육 사업으로 연계

## 3. 상세내용

### 3-1. 시스템 아키텍처

EconoBridge AI는 **Next.js 16 + Vercel AI SDK 6 + Claude Sonnet 4.6** 기반의 웹 애플리케이션으로 구현되었습니다.

```
┌─────────────────────────────────────────────────────────┐
│                    사용자 (웹 브라우저)                     │
│              "경기도 IT 개발자 채용 찾아줘"                  │
└──────────────────────┬──────────────────────────────────┘
                       │ 자연어 입력
                       ▼
┌─────────────────────────────────────────────────────────┐
│              EconoBridge AI 에이전트                       │
│         (Claude Sonnet 4.6 + ReAct 패턴)                  │
│                                                           │
│  ┌─────────┐   ┌──────────┐   ┌──────────┐               │
│  │ 질의 분석 │ → │ 도구 선택 │ → │ 결과 종합 │              │
│  └─────────┘   └──────────┘   └──────────┘               │
│        최대 10단계 자율 도구 호출 (ToolLoopAgent)            │
└──────────────────────┬──────────────────────────────────┘
                       │ API 호출
       ┌───────────────┼───────────────┐
       ▼               ▼               ▼
┌────────────┐  ┌────────────┐  ┌────────────┐
│  채용정보    │  │  안전정보    │  │  부가정보    │
│             │  │             │  │             │
│ ·워크넷     │  │ ·근로복지공단 │  │ ·HRD-Net   │
│ ·장애인고용  │  │ ·고용노동부  │  │ ·안전보건공단│

│  🛡️ 기업별 안전등급 (A/B/C/D)                              │
```

### 3-2. 7개 AI 도구 상세

AI 에이전트가 자율적으로 호출하는 7개 전문 도구는 각각 공공데이터 API와 연결됩니다.

| # | 도구명 | 데이터 출처 (기관) | 주요 기능 |
|---|--------|-------------------|----------|
| 1 | `search_worknet_jobs` | 워크넷 (한국고용정보원) | 키워드·지역·경력 기반 채용공고 검색 |
| 2 | `search_disability_jobs` | 한국장애인고용공단 (KEAD) | 장애유형·근무환경 고려 맞춤 채용검색 |
| 3 | `search_training_courses` | HRD-Net (한국산업인력공단) | K-디지털·내일배움카드 훈련과정 추천 |
| 4 | `check_company_safety` | 근로복지공단 + 고용노동부 | **기업 안전점수 산출 (핵심 기능)** |
| 5 | `check_wage_violations` | 고용노동부 | 임금체불 이력·금액·피해자 수 조회 |
| 6 | `search_safety_laws` | 한국산업안전보건공단 (KOSHA) | 관련 안전보건법령 검색·안내 |
| 7 | `generate_cover_letter` | AI 생성 (Claude) | NCS 기반 맞춤 자기소개서 자동 작성 |

### 3-3. 안전점수 산출 알고리즘 (핵심 차별화 기능)

EconoBridge AI의 가장 핵심적인 차별화 요소는 **기업 안전점수 자동 산출 시스템**입니다. 근로복지공단 산재처리현황과 고용노동부 중대산업사고 데이터를 교차 분석하여 100점 만점의 안전점수를 산출합니다.

**산출 공식:**

```
안전점수 = 100점 (기본)
  - 산업재해 건당 ············ -5점
  - 중대재해 건당 ············ -15점
  - 사망사고 건당 ············ -25점
  - 임금체불 이력 있을 시 ···· -10점
  (최저 0점, 최고 100점)
```

**안전등급 체계:**

| 등급 | 점수 범위 | 의미 | 표시 |
|------|----------|------|------|
| **A** | 90점 이상 | 안전 | 🟢 녹색 |
| **B** | 70~89점 | 보통 | 🟡 노란색 |
| **C** | 50~69점 | 주의 | 🟠 주황색 |
| **D** | 50점 미만 | 위험 | 🔴 빨간색 |

이 알고리즘은 실제 코드로 구현되어 작동하며(`lib/tools/safety-check.ts`), AI 에이전트가 채용공고에 등장하는 기업명을 자동으로 추출하여 안전점수를 조회합니다. D등급(위험) 기업은 채용 결과에서 **명확한 경고 표시**와 함께 제공됩니다.

### 3-4. 자율 에이전트 워크플로우 (ReAct 패턴)

EconoBridge AI는 Vercel AI SDK 6의 `ToolLoopAgent`를 활용하여 **최대 10단계의 자율 도구 호출**을 수행합니다. 사용자가 한 번 질문하면, AI 에이전트가 스스로 판단하여 필요한 도구를 순서대로 호출합니다.

**워크플로우 예시 (청년 구직자):**

```
[사용자] "경기도에서 IT 웹개발자 채용 찾아줘"
    │
    ▼ Step 1: 질의 분석
    프로필 추출 → 장애: 없음, 지역: 경기, 직종: IT/웹개발
    │
    ▼ Step 2: search_worknet_jobs 호출
    경기도 IT 웹개발 채용공고 5건 검색
    │
    ▼ Step 3~7: check_company_safety × 5회
    검색된 5개 기업의 안전점수 각각 조회
    │
    ▼ Step 8: check_wage_violations
    임금체불 이력 교차 확인
    │
    ▼ Step 9: search_training_courses
    역량 갭 보완 훈련과정 추천
    │
    ▼ Step 10: 종합 결과 정리
    안전점수순 정렬 + 위험기업 경고 + 훈련 안내
```

**장애인 구직자 자동 감지**: 사용자의 발화에서 장애 관련 키워드가 감지되면, AI 에이전트가 자동으로 `search_disability_jobs`(장애인 맞춤 채용)를 우선 호출하여 장애유형과 근무환경 접근성을 고려한 채용정보를 제공합니다.

### 3-5. 활용 시나리오

#### 시나리오 A: 장애인 구직자

> **입력**: "서울에서 장애인이 근무할 수 있는 사무직 채용 알려줘"

- AI가 장애인 키워드 감지 → `search_disability_jobs` 우선 호출
- 장애유형별 접근 가능 사무직 채용 5건 검색
- 각 기업 안전점수 자동 조회 → A등급(안전) 기업 우선 추천
- 필요시 장애인 직업훈련과정 연계 안내

#### 시나리오 B: 청년 구직자

> **입력**: "경기도에서 IT 개발자 채용 찾아줘, 훈련도 추천해줘"

- `search_worknet_jobs`로 경기도 IT 채용 검색
- 5개 기업 안전점수 교차검증 → 위험기업 1곳 경고
- `search_training_courses`로 K-디지털 훈련과정 3건 추천
- 안전점수 기준 정렬된 종합 결과 제공

#### 시나리오 C: 중장년 전직자

> **입력**: "인천에서 제조업 생산직 일자리 있나요?"

- `search_worknet_jobs`로 인천 제조업 채용 검색
- **제조업 특성상 산재 위험이 높으므로** 안전점수 검증이 특히 중요
- 산재이력이 있는 기업은 상세 경고와 함께 대안 제시
- 관련 안전보건법령(`search_safety_laws`) 안내

## 4. 기대효과

### 4-1. 구직자 측면

| 효과 | 설명 |
|------|------|
| **위험 일자리 사전 회피** | 산재이력·중대재해 기록이 있는 기업을 구직 단계에서 사전에 식별하여, 위험한 일자리에 취업하는 것을 예방 |
| **정보 비대칭 해소** | 분산된 공공데이터(산재·체불·안전법령)를 AI가 자동으로 통합 조회하여 구직자에게 투명한 정보 제공 |
| **취약계층 맞춤 지원** | 장애인·청년·중장년 등 취약계층별 특화된 채용검색 및 훈련과정 추천으로 고용 접근성 향상 |
| **역량 개발 연계** | 채용 요구역량과 구직자 현재 역량 간의 갭을 분석하여 맞춤 훈련과정을 함께 추천 |

### 4-2. 재단 공익사업 연계

EconoBridge AI는 산업안전상생재단의 공익사업과 다음과 같이 직접 연계될 수 있습니다:

**1) 중소기업 안전수준 가시화 → 자발적 안전개선 유도**
- 기업의 안전점수가 구직자에게 공개됨으로써, 안전점수가 낮은 기업은 **우수 인재 확보에 불리**해짐
- "안전한 기업이 좋은 인재를 얻는다"는 인식이 형성되어 기업의 자발적 안전투자 동기 부여

**2) 재단 컨설팅·교육 사업 연계**
- 안전점수 D등급(위험) 기업 → 재단의 안전컨설팅 서비스 안내 연계
- 안전점수 C등급(주의) 기업 → 재단의 안전교육 프로그램 안내 연계
- 안전점수 개선 이력 추적을 통한 재단 사업 성과 측정 지표로 활용 가능

**3) 안전문화 확산 플랫폼**
- 기업 안전정보의 대중화를 통해 산업안전에 대한 사회적 관심 제고
- 구직자·기업·재단 3자 간의 선순환 구조 형성

### 4-3. 사회적 효과
